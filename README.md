# Customer-Relationship-Management-CRM-Application

## 🚀 Project Description
This is a Spring Boot based Customer Relationship Management (CRM) application.
It provides REST APIs to manage customer data.

## 🛠️ Technologies Used
- Java 21
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- REST APIs
- Postman

## 📌 Features
- Create Customer
- Get All Customers
- Get Customer By ID
- Update Customer
- Delete Customer
- Bulk Insert Customers
- Search by First Name / Last Name
- Filter by Age

## 📬 API Endpoints

| Method | Endpoint | Description |
|--------|--------|------------|
| POST | /api/customers/insert | Add customer |
| GET | /api/customers/all | Get all |
| GET | /api/customers/{id} | Get by ID |
| PUT | /api/customers/update | Update |
| DELETE | /api/customers/{id} | Delete |

## 🗄️ Database
MySQL Database: `crm`

## ▶️ Run Project
1. Clone repo
2. Open in IDE
3. Configure MySQL in `application.properties`
4. Run Spring Boot application
